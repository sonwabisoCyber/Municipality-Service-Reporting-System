# Municipality Service Reporting System

## Project Overview
The Municipality Service Reporting System is a web application designed to help residents report municipal service issues and allow staff members to manage reported cases.

## Features Completed
Created a login page.
Created a registration page.
Added navigation from the login page to the registration page.
Designed the user interface using HTML and CSS.
Added user role selection (Resident and Staff Member).
Added email/ID and password input fields.
Created a basic registration form with:
  - Full Name
  - Email
  - Password
  - Confirm Password
  - Address/Department
 
## Technologies Used
HTML5
CSS3
